ملف التعليمات - خطوات سريعة

1) تحميل الملف jood_upload.zip وفك الضغط داخله.
2) انسخ الملفات الموجودة إلى مجلد مشروعك:
   C:\Users\jooda\OneDrive\Desktop\jood_store_final_with_index
   (أو افتح نافذة مستكشف الملفات واسحب الملفات إلى مجلد المشروع).

3) طريقة التنفيذ السريعة (مقترحة):
   - افتح CMD داخل مجلد المشروع (Shift + Right Click -> Open PowerShell window here أو افتح CMD وادخل المسار).
   - انسخ والصق أوامر الملف git_commands.txt خطوة بخطوة، أو شغّل ملف apply_gitignore.bat (انقر مزدوج).

4) ملاحظة أمان:
   - ملف apply_gitignore.bat يقوم بإنشاء .gitignore محلياً لكنه لا يقوم بتنفيذ git push تلقائياً.
   - تأكد أنك قمت بتسجيل الدخول إلى GitHub وتهيئة git قبل تنفيذ أوامر git push.

إذا احتجت أعدل أي شيء أو أرفق ملفات إضافية (مثل README مفصّل)، أخبرني.
